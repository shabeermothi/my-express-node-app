# my-express-node-app

A Node Express webapp scaffolded with Yeoman.

To clone the repo

```bash
git clone https://github.com/shabeermothi/my-express-node-app.git
```

To build the webapp

```bash
npm install
```

To run the webapp
```bash
grunt
```

You can run the app directly through Node as well
```bash
node ./bin/www
```
